## 🌟 About | Acerca de

**KONTACT VO** is a professional B2B automotive marketplace platform designed for used vehicle trading, auctions, and business networking. Our platform connects automotive professionals worldwide, creating opportunities and building the future of automotive business.

**KONTACT VO** es una plataforma profesional B2B de marketplace automotriz diseñada para el comercio de vehículos usados, subastas y redes de negocio. Nuestra plataforma conecta profesionales automotrices a nivel mundial, creando oportunidades y construyendo el futuro del negocio automotriz.

### 🎯 Mission | Misión

*Connecting Professionals, Creating Opportunities, Building the Future of Automotive Business for Used Vehicles*

*Conectando Profesionales, Creando Oportunidades, Construyendo el Futuro del Automotive Business del Vehículo de Ocasión*

## 🛠️ Technology Stack | Stack Tecnológico

- **Frontend:** React 18 + TypeScript + Vite
- **Styling:** Tailwind CSS + shadcn/ui components
- **Backend:** Supabase (PostgreSQL + Edge Functions)
- **Authentication:** Supabase Auth with role-based access
- **File Storage:** Supabase Storage with security policies
- **Real-time:** Supabase Realtime subscriptions
- **Deployment:** Lovable Platform

## 🚀 Quick Start | Inicio Rápido

### Prerequisites | Prerrequisitos

- Node.js 18+ and npm
- Git
- Supabase account

### Installation | Instalación

```bash
# Clone the repository | Clonar el repositorio
git clone <YOUR_GIT_URL>
cd kontact-vo

# Install dependencies | Instalar dependencias
npm install

# Copy environment file | Copiar archivo de entorno
cp .env.example .env

# Configure your environment variables | Configurar variables de entorno
# Edit .env with your Supabase credentials

# Start development server | Iniciar servidor de desarrollo
npm run dev
```
